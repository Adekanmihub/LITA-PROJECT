### Project Overview
This aims at creating my first dashboard tracker which tells the story of attrition rate in a company, with this dashboard we are able to see the total number of employee, current employee and attrition rate showing a breakdown of attrition rate by age , by depatrment , by educational field, by gender and by age.

In this project we could filter by by department using slicer feature on the power  Bi dashboard, with this filter, we know  the attrition rate by each department in the company and track the rate at which employees leave the company and reason.
